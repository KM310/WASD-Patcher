# Security Policy

## Reporting a Vulnerability

If you discover a bug, crash, exploit, or any behavior that could be considered a security issue, please report it responsibly.

### How to Report
Do NOT open a public GitHub issue for security-related problems.

Instead, contact me directly on Discord:  
**`k_m0310`**

### What to Include
- A clear description of the issue
- Steps to reproduce it
- Why it may be a security concern
- Optional: screenshots or logs

## Supported Versions
This project is maintained on a best-effort basis.  
Security fixes will be applied to the latest version only.

## Responsible Disclosure
Please give me reasonable time to investigate and fix the issue before sharing it publicly.
